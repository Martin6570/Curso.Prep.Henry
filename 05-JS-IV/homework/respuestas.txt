Objetos ............: son contenedores donde se pueden almacenar datos, sin necesidad de depender de un indice numerico
                      como en los arreglos. se pueden nombrar de cualquier nombre cada item(clave) y almacenar cualquier dato (valor).

Propiedades ........: propiedades es el nombre que recibe el conjunto de clave y valor de un objeto. 
Métodos ............: cuando adentro de un objeto, el valor asignado a una clave, es una funcion, se le llama Metodo
Bucle for…in .......: es un bucle especial, para usar con objetos, que como no tienen indice numerico, se necesita este bucle para poder recorrerlo

Notación de puntos vs notación de corchetes: la notacion de puntos es mas rapida, pero la notacion de corchetes, permite invocar funciones
